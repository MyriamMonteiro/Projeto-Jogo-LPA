Este projeto consiste em um jogo simples desenvolvido em Python com a biblioteca Pygame, criado como parte de estudos em lógica de programação para o curso de Análise e Desenvolvimento de Sistemas (ADS). A mecânica e o tema do jogo foram inspirados na minha própria cachorrinha, chamada Kitty.

Abaixo, os principais pontos técnicos do projeto:

Objetivo: O jogador controla a Kitty e deve coletar os itens de carne que caem do céu para somar pontos, evitando os alfaces, que diminuem a vida.

Design Visual: Toda a parte gráfica, incluindo o cenário e as ilustrações da personagem e dos itens, foi desenvolvida e editada no Adobe Photoshop.

Funcionalidades: O jogo possui uma tela de menu com instruções, trilha sonora de fundo e uma tela de encerramento que indica vitória ou derrota.

Controles: A movimentação é feita através das setas laterais do teclado e a interação com os menus é realizada pela tecla espaço.
